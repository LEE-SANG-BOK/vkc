제목: E‑7 전환 3개월 액션 플랜(유학생용 1페이지 체크리스트)

빠른 적합도 셀프체크
- 학위 전공 ↔ 지원 직무가 구체적으로 연결되는가? (업무기술서 기준 키워드 5개 매칭)
- 학위 취득일 기준 3개월 내 취업활동 계획서·고용확약서 제출 가능 여부
- 학위 이후 정규 고용 근거 보유 여부(인턴·아르바이트는 원칙적으로 불인정)
- ICT/이공계 분야 지원 여부(상대적 유리)

12주(3개월) 액션 플랜
- 주 1–2: 목표 직무·직무기술서 수집, 전공·프로젝트와의 연계 문장 설계
- 주 3–4: 취업활동 계획서 초안 작성, 포트폴리오/프로젝트 증빙 정리
- 주 5–6: 타깃 기업 컨택, 고용확약서 요청/조율, 외활허가 체크(근로 시)
- 주 7–8: 기업 고용사유서 초안 가이드 공유, 표준근로계약 체크포인트 전달
- 주 9–10: 서류 일괄 검토(48시간), 보완자료(자격증·프로젝트) 첨부
- 주 11–12: 최종 제출·상태 모니터링, 추가소명 대비 Q&A 준비

필수 서류(유학생 관점)
- 취업활동 계획서(12주 로드맵·주차별 액션·KPI)
- 고용확약서(직무·연계성·훈련계획·입사 예정일)
- 학위증명·성적증명(영문/국문), 어학 성적(TOPIK 등)
- 프로젝트/경력 증빙(졸업 후 정규 경력 중심, 인턴/아르바이트는 원칙적 불인정)
- 표준이력서/경력기술서(직무기술서 키워드 반영)

직무연관성 입증 패키지
- 전공 과목→직무 역량 매핑(3–5개) | 예: 데이터구조→백엔드 API
- 프로젝트 2–3건 요약(문제/역할/성과/적용기술/직무 연결 문구)
- 자격증·수료증(직무 핵심 역량 보완)

리스크 관리
- 데드라인: 학위일+3개월 초과 시 전환 난이도↑
- 경력: 인턴·아르바이트만 보유 시 불인정 가능성 높음→프로젝트·자격증로 보완
- 근로: 외활허가 없이 근로/시간 초과는 제재 리스크
