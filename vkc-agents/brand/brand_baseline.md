# 브랜드 이미지/신뢰 기준선

톤 & 매너
- 사실 기반·친근·투명. 과장 금지, 조건/확률 명시.
- 한-베 병기, 문화 코드 반영(밈은 양념 수준).

시각 요소
- 인증 배지 상단 고정 권장(히어로/신뢰섹션), 채택 사례 요약(근거 확보).
- 카드뉴스 5장 구조, 쇼츠 큰 자막·고대비.

신뢰 장치
- 면책: “경험 공유, 법률 자문 아님” 상시 노출.
- 사회적 증거: 숫자/리뷰/저장 수를 카피에 반영.
- 데이터 투명: 09:00 리포트로 성과 공개 가능 범위 공유.
- 리스크 제로: 필요 시 환불/보장 문구로 주저함 제거(정책 수립 시 적용).

이중 자막/오디오
- 모든 숏폼/카드에 KR/VI 자막 병기(가독 폰트/색 구분). 무음 시청 대비 필수
- 오디오는 콘텐츠 톤과 일치. 트렌딩 사운드 활용하되 과도한 소음/불일치 금지

AI/버추얼 아바타 가이드
- AI/CGI/보이스오버 사용 시 투명 라벨 표기(“AI 생성/보이스” 등)
- 품질>유명세 원칙: 얼굴 노출 없이도 정보·연출 퀄리티 우선
- 과장/오인 유발 편집 금지. 콘텐츠 출처/근거 명시

카피 가이드 예시
- 권위형: “검증된 답변, 관리자 인증 완료” / “24시간 내 1차 응답 목표”
- 숫자형: “벌써 N명이 선택” / “저장률 X% 콘텐츠”
- 솔직형: “익명 정보는 부족합니다. 그래서 인증을 도입했습니다.”
