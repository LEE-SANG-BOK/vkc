# Google Forms 문항 텍스트(KR/VI)

A) 사전 수요 조사 (12문항)
1) 연령대 / Độ tuổi: 18–24 / 25–29 / 30+
2) 거주지 / Nơi ở: Việt Nam / Hàn Quốc / Khác
3) 관심 비자/목적 / Mục tiêu: E‑7‑4 / D‑2 / F‑6 / Việc làm / Khác
4) 주요 정보 채널 / Kênh thông tin: Facebook / Zalo / TikTok / YouTube / Khác
5) 가장 큰 고민 / Khó khăn lớn nhất: 절차 / 서류 / 비용 / 사기 / Khác
6) 정보 신뢰 기준 / Tiêu chí tin cậy: 검증된 경험 / 관리자 인증 / 조회수 / Bạn bè
7) 사기/오해 경험 / Từng gặp lừa đảo/hiểu lầm?: 예/아니오 (+사례 선택)
8) 예산·시간 제약 / Hạn chế ngân sách/thời gian: 낮음/중간/높음
9) 긴급도 / Mức độ gấp: 즉시 / 1개월 / 3개월 / 탐색 중
10) 선호 답변 방식 / Cách nhận câu trả lời: 텍스트 / 영상 / 체크리스트
11) 알림 채널 / Kênh thông báo: Email / Zalo / Facebook DM
12) 오픈엔드 / Câu hỏi mở: 최근 가장 혼란스러웠던 질문 1가지

B) 온보딩 마이크로 설문 (5문항)
1) 유입 채널(utm_source와 일치) / Kênh vào: fb/zalo/tt/yt/khác
2) 기대 기능 / Tính năng mong đợi: 검증답변 / 빠른응답 / 커뮤니티 / 자료실
3) 첫 질문 의향 / Ý định đặt câu hỏi đầu tiên: 1–5
4) 인증 가치 인식 / Nhận thức về “câu trả lời đã xác thực”: 1–5
5) 자유 의견 / Ý kiến khác (tùy chọn)

C) NPS (3문항)
1) 0–10 추천 의향 / Mức độ giới thiệu 0–10
2) 추천/비추천 이유 / Lý do
3) 개선 1가지 / Một cải tiến

오프라인 경량(5–7)
- 현재 준비 / Khâu đang chuẩn bị
- 가장 어려운 점 / Khó khăn lớn nhất
- 신뢰 기준 / Tiêu chí tin cậy
- 이용 의향(5점) / Ý định sử dụng(1–5)
- 연락 동의(선택) / Đồng ý nhận liên hệ (tùy chọn)
