# Google Forms 문항 텍스트(KR/VI)

A) 사전 수요 조사 (15문항)
1) 연령대 / Độ tuổi: 18–24 / 25–29 / 30+
2) 거주지 / Nơi ở: Việt Nam / Hàn Quốc / Khác
3) 현재 단계 / Tình trạng hiện tại: 준비 중 / 한국 체류 / 귀국 후 재도전 / 기타
4) 관심 비자·목적 / Mục tiêu: D‑10 연장 / E‑7‑1 / E‑7‑4 / F‑6 / 인턴·정규직 / Khác
5) 직무·산업 관심 / Lĩnh vực quan tâm: 마케팅/AE / 글로벌 영업 / IT / 제조/품질 / 기타 (복수 선택)
6) 가장 큰 고민 / Khó khăn lớn nhất: 절차 / 고용사유서·계약 / 비용 / 사기 / 시간 / Khác
7) 정보 신뢰 기준 / Tiêu chí tin cậy: 검증된 경험 / 관리자 인증 / 공식 공지 / 조회수 / Bạn bè
8) 사기/오해 경험 / Từng gặp lừa đảo/hiểu lầm?: 예/아니오 (+사례 선택)
9) 예산·시간 제약 / Hạn chế ngân sách/thời gian: 낮음/중간/높음
10) 긴급도 / Mức độ gấp: 즉시 / 1개월 / 3개월 / 탐색 중
11) 선호 답변 형식 / Cách nhận câu trả lời: 텍스트 / 영상 / 체크리스트 / 1:1 상담
12) 선호 채널 / Kênh ưu tiên: Facebook / TikTok / Instagram / YouTube / Email / Zalo
13) 채용·커리어 정보 필요 여부 / Có cần thông tin tuyển dụng? 예/아니오 (예 시 롯데/기업 선택)
14) 알림 채널 / Kênh thông báo: Email / Zalo / Facebook DM / WhatsApp
15) 오픈엔드 / Câu hỏi mở: 최근 가장 혼란스러웠던 질문 1가지 + 받고 싶은 자료

B) 온보딩 마이크로 설문 (6문항)
1) 유입 채널(utm_source와 일치) / Kênh vào: fb/zalo/tt/yt/ig/email/khác
2) 기대 기능 / Tính năng mong đợi: 검증답변 / 빠른응답 / 채용알림 / 커뮤니티 / 자료실
3) 첫 질문 의향 / Ý định đặt câu hỏi đầu tiên: 1–5
4) 고용사유서·체류 로드맵 필요성 / Có cần mẫu lý do tuyển dụng? 1–5
5) 인증 가치 인식 / Nhận thức về “câu trả lời đã xác thực”: 1–5
6) 자유 의견 / Ý kiến khác (tùy chọn)

C) NPS (3문항)
1) 0–10 추천 의향 / Mức độ giới thiệu 0–10
2) 추천/비추천 이유 / Lý do
3) 개선 1가지 / Một cải tiến

오프라인 경량(6)
- 현재 준비 / Khâu đang chuẩn bị
- 가장 어려운 점 / Khó khăn lớn nhất
- 직무·채용 관심 / Lĩnh vực việc làm quan tâm
- 신뢰 기준 / Tiêu chí tin cậy
- 이용 의향(5점) / Ý định sử dụng(1–5)
- 연락·채용 알림 동의(선택) / Đồng ý nhận liên hệ & thông báo tuyển dụng
