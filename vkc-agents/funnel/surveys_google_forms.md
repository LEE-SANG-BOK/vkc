# 설문 템플릿 (Google Forms)

본 프로젝트에서 바로 사용 가능한 설문 구조입니다. KR/VI 병기 권장.

## A) 사전 수요 조사 (12문항, 분기형)
1. 연령대 (18–24 / 25–29 / 30+)
2. 거주지 (베트남 / 한국 / 기타)
3. 관심 비자/목적 (E-7-4 / D-2 / F-6 / 취업 / 기타)
4. 주요 정보 채널 (Facebook / Zalo / TikTok / YouTube / 기타)
5. 최근 가장 큰 고민 (절차 / 서류 / 비용 / 사기 위험 / 기타)
6. 정보 신뢰 기준 (검증된 경험 / 관리자 인증 / 조회수 / 지인추천)
7. 사기/오해 경험 여부 (예/아니오) + 있었다면 사례(선택)
8. 예산/시간 제약 (낮음/중간/높음)
9. 긴급도 (즉시/1개월 내/3개월 내/정보 수집 단계)
10. 선호 답변 방식 (텍스트 Q/A / 영상 요약 / 체크리스트)
11. 알림 채널 (Email / Zalo / Facebook DM)
12. 오픈엔드: 최근 가장 혼란스러웠던 질문 1가지

분기 로직 예시
- 선택한 페르소나(E-7-4/D-2/F-6)에 따라 준비도 3~5항 추가(서류/언어/재정/경력 등)

## B) 온보딩 마이크로 설문 (5문항)
1. 유입 채널 (utm_source 기준)
2. 기대 기능 (검증 답변 / 빠른 응답 / 커뮤니티 / 자료실)
3. 첫 질문 의향 (높음~낮음, 5점 척도)
4. 인증 답변 가치 인식 (높음~낮음, 5점 척도)
5. 자유 의견 (선택)

## C) NPS (3문항)
1. 추천 의향 (0–10)
2. 그 이유는 무엇인가요? (서술)
3. 개선 1가지 제안 (서술)

## D) 오프라인 경량 설문 (5–7문항, 3분 이내)
- 장소: 캠퍼스/카페/모임. QR 또는 태블릿 권장. 소정의 보상.
1. 현재 준비 중인 것 (유학/취업/비자/기타)
2. 가장 어려운 점 (절차/서류/비용/사기/기타)
3. 정보 신뢰 기준 (검증된 경험 / 관리자 인증 / 기타)
4. 서비스 이용 의향 (5점 척도)
5. 연락/알림 수단 동의 (선택)
6. 오픈엔드 1문항(선택)

## 운영 팁
- 질문은 짧고 명확하게, 객관식 위주
- KR/VI 이중 언어
- 설문 종료 후 요약 결과를 커뮤니티에 공유(신뢰/참여 상승)
