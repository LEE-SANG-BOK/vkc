# 오프라인 설문 표본 설계 (3분 경량)

목표 표본
- 총 100+명 (가능 시 120~150)
- 성별: 남 50% / 여 50%
- 연령: 20–24 50% / 25–29 50%
- 직군: 학생/직장인 균형(현장 여건 반영)

쿼터 예시(각 20–30명)
- 20대 초반 남학생 / 후반 남직장인 / 초반 여학생 / 후반 여직장인

문항(5–7개)
- 현재 준비(유학/취업/비자/기타), 가장 어려운 점(절차/서류/비용/사기/기타)
- 신뢰 기준(검증된 경험/관리자 인증/기타), 이용 의향(5점), 연락 동의(선택), 오픈엔드(선택)

운영 팁
- 보상 소액 지급(즉시), QR로 LP·설문 연결, 현장 데모 가능 시 가입 유도
