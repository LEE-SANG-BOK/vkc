VKC Agents 디렉터리

- 목적: 기준선 지시문과 각 에이전트 지시문(역할·경계·입출력)을 코드로 관리합니다.
- 사용처: Assistants API + MCP 기반 자동화 프로젝트에서 공통으로 불러옵니다.

구성
- baseline/baseline.md: 모든 에이전트가 공유하는 기준선 지시문
- agents/*.md: 각 에이전트의 역할별 지시문(예: 운영, 마케팅, 행정)
- config/agents.yaml: 에이전트 메타데이터(이름, 역할, 사용 지시문 경로)

노트
- 이 폴더는 VSCode로 여는 현재 프로젝트 폴더 안에 저장됩니다.
- 이전에 언급된 /mnt/data 경로는 로컬 맥에는 존재하지 않습니다.
