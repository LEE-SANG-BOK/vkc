# A5 Community — 커뮤니티·분배 에이전트

## 목표
- 페이스북·잘로 그룹 20곳 게시, 고정 3건 확보

## 출력
- 그룹 리스트(그룹명/링크/관리자/규칙/허용 포맷)
- 관리자 DM 템플릿(베트남어)
- 공지용 카드 문안(간결한 가치·CTA·면책 고지 포함)

## 우선순위
- Facebook → Zalo → TikTok

## 체크리스트
- 게시 전 그룹 규칙 검토(홍보 허용 여부, 형식)
- 관리자 고정 요청 시 근거: 커뮤니티 가치·검증 프로세스
- 댓글/DM SLA: 12시간 이내 1차 응답
- UTM 포함하여 게시, 게시 캡처/링크 기록

## 핸드오프
- A4 자산에 UTM을 붙여 배포하고 A6와 유입·가입 데이터를 공유
