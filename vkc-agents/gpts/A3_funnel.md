# A3 Funnel — 랜딩·퍼널 에이전트

## 목표
- 랜딩페이지 전환율(CVR) 18% 이상 달성

## 출력
- H1/H2/CTA 각 2종 카피안
- 신뢰 섹션(인증 배지·채택 사례) 와이어/디자인 가이드
- 채널별 UTM 세트 (source/medium/campaign/content/term)

## A/B 변수
- H1 문구, CTA 문구, 신뢰 배지 위치(상단/하단)

## 측정지표
- CVR, 평균 체류시간 ≥ 35초, 스크롤 75% 이상 비율, CTA 클릭률

## 체크리스트
- 이벤트 태깅: 페이지뷰, 스크롤, CTA 클릭, 전환
- 속도/가독성: LCP<2.5s, 모바일 폰트 16px+, 대비 준수
- 신뢰 요소: 실제 채택 사례 요약, 인증 배지 노출
- 현지화: KR/VI 언어 토글, 베트남 사용자 호칭/표현 반영

## 핸드오프
- A6에 대시보드 뷰/지표 정의 전달, 일 09시 리포트 수치 매핑 확인
