# 100만원 예산 운용안

배분(권장)
- 콘텐츠/도구: ~30만원 (카드뉴스/영상 툴, Sora2)
- 프로모션/인센티브: ~20만원 (설문 보상, 경품)
- 온라인 광고: ~30만원 (소액 A/B, 성과 채널 증액)
- 예비비: ~20만원 (리스크/기회 대응)

원칙
- 소액 실험 → 승자 증액, 패자 중지
- 커뮤니티/UGC/밈 활용으로 유기 도달 극대화
- 설문/오프라인 접점으로 리드 확보 병행

슈퍼팬(10명) 전략
- 초기 핵심 사용자 10명 선발(그룹/커뮤니티 활동성 기반)
- 인센티브 일부 배정(소액 굿즈/이벤트/역할 부여), UGC/후기 증폭
- 추천/초대 코드로 자연 확산 유도(추후 시스템 연동)

성과 지표(예)
- CTR≥2.5%, LP CVR≥18%, 가입당 ≤1,200원(초기 가설)
