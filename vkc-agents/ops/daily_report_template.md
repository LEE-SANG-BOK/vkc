# 09:00 데일리 퍼널 리포트 템플릿

보고 시각: 09:00 (전일 데이터 기준)

## 1) 요약 (3줄)
- 핵심: 방문 → 가입 → 첫 질문 → 첫 답변 → 채택 추이
- 전일 대비 증감, 주요 채널/콘텐츠 하이라이트
- 다음 액션 3개

## 2) 퍼널 지표
- 방문수: N
- 가입수: N (CVR = 가입/방문 %)
- 첫 질문수: N
- 첫 답변수: N (SLA 평균: xxh)
- 채택수: N (채택률: 채택/답변 %)

## 3) 채널별 요약 (utm_source)
- fb: 방문 N / 가입 N / CVR % / CTR % / 체류 sec / 저장률 %

## 4) 테스트별 성과 (A/B)
- G1(H1): h1_a vs h1_b → CVR %, 체류 sec
- G2(Hook): hook_a vs hook_b → 3초 이탈률, VTR
