# 콘텐츠 제작 흐름 권장(Shorts → Card News)

우선순위
- 숏폼 영상 선제작(알고리즘 도달 극대화) → 카드뉴스로 재가공(그룹/피드 확장)

흐름
- 아이디어/스크립트(템플릿 T1~T6) → 숏폼 제작(20–35s, KR/VI 자막, 훅/CTA) → 성과 확인
- 성과 상위 영상 요약 → 카드뉴스 3–5장(핵심 팁/체크/CTA) → 그룹 게시·저장 유도

장점
- 실제 반응으로 주제 검증 → 고효율 리소스 배분
- 동일 주제로 영상+이미지 2채널 커버(UGC/공유 촉진)

유의
- 영상→카드 전환 시 스토리의 맥락 유지, 밈은 로컬화
- 워터마크 제거본 사용(타 플랫폼 교차 업로드 페널티 방지)
