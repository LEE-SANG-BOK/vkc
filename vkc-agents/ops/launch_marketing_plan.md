# 가입자 확보 중심 초기 마케팅 전략 (VKC)

목표
- 1개월 내 가입자 목표: 현실치 N1–N2, 극성 팬 10명+ 육성
- 7일 베타 목표: 가입 120±50, 첫 질문 40±15, 채택 15±8
- 메시지 기준: “검증된 선경험자 + 관리자 인증” 일관 적용

예산(100만원 내) 배분 안
- 콘텐츠/도구: ~30만원 (카드뉴스/영상 툴, Sora2)
- 프로모션/인센티브: ~20만원 (설문 보상, 경품)
- 온라인 광고: ~30만원 (소액 실험·확인)
- 예비비: ~20만원 (성과 채널 증액·리스크 대비)

랜딩페이지 구성 (A/B 병행)
- H1/H2/CTA: 2안 운영 (검증된 선경험자 vs 관리자 인증)
- 신뢰요소: 인증 배지, 채택 사례, 면책 고지(상/하단 위치 테스트)
- 퍼널 태깅: utm_campaign=d-5_launch, utm_content=h1_a|h1_b, 이벤트(page_view/scroll_75/cta_click/signup)

채널별 운영안(비광고 중심)
- Facebook 그룹: 20곳 리스트업, 고정 3건 목표. 관리자 DM(베트남어) 템플릿 사용.
- Zalo OA/DM: 1:1 문의·알림 보조, 설문/LP 링크 공유.
- TikTok/YouTube Shorts: 주 6편 수준, 훅/완시율 집중, 프로필 링크로 LP 유도.
- 보조: 블로그/브런치·커뮤니티(에브리타임/카페) 정보형 게시.

숏폼 가이드(Sora2)
- 길이 20–35초, 첫 3초 훅, 한-베 자막, 해시태그/트렌딩 사운드.
- 템플릿: T1 체크리스트, T2 오해→팩트, T3 미니 팁 등.

오프라인 설문(표본 설계)
- 목표 표본 100+: 성별 1:1, 20–24 vs 25–29 균형, 학생/직장인 쿼터.
- 경량 5–7문항(3분 내), 즉시 보상 소액 지급, QR로 LP 유도.

브랜드/신뢰 기준
- 톤: 사실 기반·친근, 과장 금지. 면책 고지 상시.
- 사회적 증거: 수치/리뷰/저장 수 강조, 데이터 투명 공유.
- 배지/인증: 시각 일관성 유지(상단 고정 권장), 증빙 즉시 파기 고지.
